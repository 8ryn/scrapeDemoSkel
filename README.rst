webScrapeBBCSkel
===========================

|code_ci| |docs_ci| |coverage| |pypi_version| |license|

A module which reads the maximum and minimum forecast temperatures from BBC
Weather and pushes them to a soft IOC. Just to learn how to use this skeleton!
